# Combinations
Напишете функция, която изчислява колко решения в естествените числа (включително и нулата) има уравнението:
<br>x1 + x2 + x3 = n
<br>Числото n е цяло число и се въвежда от конзолата. 

<br>Примерен вход и изход
| вход     | изход |
|----------|-------|
| (["25"]) | 351   |

| вход     | изход |
|----------|-------|
| (["20"]) | 231   |

| вход    | изход |
|---------|-------|
| (["5"]) | 21    |
